# Chikauak (Backend)

> **Status:** Under Development

A lightweight, 100% serverless backend API designed to log and manage gym workout sessions.
What *Chikaukak* means? [Take a look](https://gdn.iib.unam.mx/diccionario/chicahuac/181743)

## What it Uses (Tech Stack)

*   **AWS Lambda**: Executes the core backend business logic.
*   **Amazon DynamoDB**: Stores workout logs with low latency.
*   **Language**: Java 17
